คำถามที่พบบ่อย

ครอบคลุมประเด็นธุรกิจ การรวมระบบ ความปลอดภัย และราคา

ระบบรองรับธนาคารอะไรบ้าง? +
รองรับธนาคารหลักในประเทศไทย และจะเพิ่มรายการอย่างต่อเนื่อง รายละเอียด endpoint ดูได้ในเอกสาร API
ข้อมูลปลอดภัยแค่ไหน? มีมาตรการอะไรบ้าง +
เข้ารหัส TLS, เก็บบันทึกกิจกรรม (Audit log), RBAC, IP allowlist และมี Webhook signature เพื่อตรวจสอบความถูกต้องของข้อมูล
การคิดค่าบริการเป็นอย่างไร? +
คิดตามจำนวนคำขอต่อเดือน มีแผน Free/Pro/Business และรองรับปริมาณสูงพร้อม SLA สำหรับองค์กร
เชื่อมต่อยากไหม? มี SDK หรือไม่ +
เชื่อมง่ายผ่าน REST/JSON มีตัวอย่างโค้ด (Node.js/cURL) และ Sandbox สำหรับทดสอบ รวมถึง Webhook แจ้งผลย้อนหลัง
เก็บข้อมูลสลิปไว้นานแค่ไหน? +
กำหนดระยะเวลาการเก็บตามแพ็กเกจ (เช่น 30–180 วัน) และสามารถร้องขอลบ/ทำให้ไม่สามารถระบุตัวตนได้ตามนโยบายความเป็นส่วนตัว

บริการ API ตรวจสอบสลิปโอนเงิน & บอทเช็คสลิป ช่วยให้ธุรกิจของคุณมั่นใจได้ทุกธุรกรรม
