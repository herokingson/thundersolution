เช็คสลิปปลอม/สลิปซ้ำ แบบอัตโนมัติ

บริการ API สำหรับธุรกิจที่ต้องการยืนยันธุรกรรมโอนเงินอย่างรวดเร็วและปลอดภัย ลดเคสสลิปปลอม ประหยัดเวลาทีมงาน และเพิ่มความเชื่อมั่นให้ลูกค้า

Uptime 99.9%
เฉลี่ย < 500 ms/คำขอ
TLS + Webhook Signature

เชื่อมง่าย เข้ากับสแต็กของคุณ

REST/JSON + Webhook + ตัวอย่างโค้ดหลายภาษา

การยืนยันตัวตน

ใช้ Authorization: Bearer <API_KEY> ในทุกคำขอผ่าน HTTPS (TLS) 

curl -H "Authorization: Bearer <YOUR_API_KEY>" https://api.thunder.in.th/v1/health

Endpoints

POST
/v1/slip/verify

ตรวจสอบสลิปจากรูปภาพหรือข้อมูลธุรกรรม

{
    "bank": "kbank",
    "transferred_at": "2025-08-21T10:15:33+07:00",
    "amount": 590.00,
    "ref_no": "KBANK-102233",
    "slip_image": "<base64>"
  }
POST
/v1/slip/ocr

ดึงข้อความ/ข้อมูลจากภาพสลิป (OCR)

{
    "slip_image": "<base64>"
  }
GET
/v1/transactions/:id

ดูผลตรวจธุรกรรมย้อนหลัง

POST
/v1/webhooks/test

ยิงทดสอบ Webhook เพื่อเช็คการรับส่ง

ความปลอดภัย

  • ทุกคำขอผ่าน TLS
  • Webhook มีการลงลายเซ็น (HMAC) ตรวจสอบฝั่งผู้รับ
  • รองรับ IP allowlist และ Role-based Access (ตามแพ็กเกจ)
รองรับการแจ้งผลผ่าน LINE/Discord/Telegram (บอท)
